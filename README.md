# codealpha_tasks
## 1. Credit Scoring Model
#### Description

#### A machine learning project that predicts an applicant's creditworthiness using historical financial data. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation using classification algorithms such as Logistic Regression, Decision Tree, Random Forest, and XGBoost to support data-driven credit risk assessment.

#### 2. Disease Prediction from Medical Data
#### Description

#### A machine learning project that predicts the likelihood of diseases such as Heart Disease, Diabetes, and Breast Cancer using structured medical datasets. The project implements data preprocessing, exploratory data analysis (EDA), feature engineering, and classification algorithms including SVM, Logistic Regression to enable accurate and early disease prediction.
